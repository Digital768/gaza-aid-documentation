## Get code from azure

<Steps>
1.  Open <a target='_blank' href="https://dev.azure.com/even-matpash/_git/gaza-aid"></a> in azure.

2. Click the clone button at the top right of the page clone the code to the local computer. <Image src={cloneProject} alt={"Reports Page"}/>

3. Click Clone in VS Code. <Image src={cloneInVsCode} alt={"Reports Page"}/>

4. Open the project in the local computer.

</Steps>

## Deployment

<Steps>
1. Push changes the the azure repository.

2. In the 
<a target='_blank' href="https://dev.azure.com/even-matpash/gaza-ai">azure devops</a> 

3. Select the appropriate node pipeline.

4. Click the run pipeline button. 

5. Select the appropriate branch and click the Run button at the bottom right. 

6. in the azure web app go to Settings > Configuration and set the startup command to:
"PYTHONPATH="/home/site/wwwroot/.python_packages/lib/site-packages" python3 -m gunicorn --bind 0.0.0.0:8000 --timeout 600 --worker-class uvicorn.workers.UvicornWorker main:app".
we do it because unless we set the PYTHONPATH, it doesn't find the packages and therefore won't work.


</Steps>
