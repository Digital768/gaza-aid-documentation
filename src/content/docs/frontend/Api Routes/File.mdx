### sendUploadExcel


###### This function sends a request to the backend with an excel file and the relevant date range for data uploading.  

<Code
  title="sendUploadExcel function"
  lang="json"
  code={`
        export async function sendUploadExcel(
        file: File,
        startDate: Date,
        endDate: Date
      )
    `}
/>

#### Expected Input
  
  The function recieves three arguments: 
  1. file from type File
  2. startDate from type Date
  3. endDate from type Date

#### Expected Output

  The function returns the response status and the error if occured.
